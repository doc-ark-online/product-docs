# 资源报错问题排查指南

## 报错内容的含义与排查方向

| 报错信息             | 截图                                                    | 原因                                                         | 处理方式                                                     |
| -------------------- | ------------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 上传OSS失败          | ![img](https://arkimg.ark.online/1762944854913-37.webp) | 网络问题无法上传或者文件打包压缩失败导致，常见于文件路径过长或文件路径中存在特殊符号 | 检查是否使用代理，检查工程路径是否过长或者路径中存在特殊符号，将工程移动到较短且不存在中文字符的路径后再次上传 |
| 上传Sha1与Size不一致 | ![img](https://arkimg.ark.online/1762944854913-38.webp) | 导入的【资源A】，因为种种情况（比如某个依赖丢了）导致上传时已经变成【资源B】了，A和B文件校验匹配不上就会提示这个错误 | 新建工程，重新导入该资源，若仍然失败，请获取Mw.log,联系@陶澍卿 发送上传失败的资源 |
| Code返回值非200      | ![img](https://arkimg.ark.online/1762944854913-39.webp) | 无法正确连接至服务器，一般是网络原因或者当前上传使用的编辑器版本过于老旧 | 关闭网络代理，更新到最新版本编辑器                           |
| {0}文件错误或丢失    | ![img](https://arkimg.ark.online/1762944854913-40.webp) | 资源的依赖文件被删除，导致引用为空                           | 获取Mw.log,联系@陶澍卿                                       |
| HTTP请求失效或无效   | ![img](https://arkimg.ark.online/1762944854913-41.webp) | 无法正常发送HTTP请求，一般是网络原因导致的发送失效           | 关闭代理，切换当前网络                                       |
| 文件保存失败         | ![img](https://arkimg.ark.online/1762944854913-42.webp) | 常见于资源路径或者工程路径过长，导致文件压缩失败             | 检查工程路径是否过长或者路径中存在特殊符号，将工程移动到较短且不存在中文字符的路径后再次上传 |
